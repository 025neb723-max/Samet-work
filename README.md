# Nepal Survey Dashboard

A large-scale survey dashboard showing the **Top 50 Preferred** and **Top 50 Not Preferable** things in Nepal.

## Features

- 📊 **Bar chart** visualization of top 10 items by score
- 📋 **Sortable table** of all 50 items with rank, category, score, votes, and trend
- 🔍 **Search** items by name or category
- 🏷️ **Filter** by category and trend direction
- 📱 **Responsive** design for mobile and desktop
- 🔄 **Tab switching** between Preferred and Not Preferable lists

## How to Run

Open `index.html` in any modern browser, or use a local server:

```bash
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## Survey Stats

- **52,480** respondents across all 7 provinces and 77 districts
- Survey period: January 2025 – March 2026
- Margin of error: ±1.2%
